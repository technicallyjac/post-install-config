<h1>osTicket - Post-Install Configuration</h1>
This tutorial outlines the post-install configuration of the open-source help desk ticketing system osTicket.<br />


<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)

<h2>Operating Systems Used </h2>

- Windows 10</b> (22H2)

<h2>Post-Install Configuration Objectives</h2>

- Configure Roles
- Configure Departmennts
- Configure Teams
- Configure Agents
- Configure Users
- Configure SLA
- Configure Help Topics

<h2>Configuration Steps</h2>

Under "Admin" we will be executing post-install confiiguration.

<p>
Configure Roles by selecting Admin panel > Agents > Roles  > Add New Role. Give permissions to new role. 
</p>
<br />

<p>

</p>
<p>
Configure Departments from the Admin panel by selecting Agents > Departments > Add New Departmen. Rename your department and select "Create Dept" to confirm. 
</p>
<br />

<p>
<img width="959" alt="Screenshot 2023-07-04 at 11 12 08 AM" src="https://github.com/technicallyjac/post-install-config/assets/138219436/028e0822-bf99-42fa-9abd-6fe5eb8e9f07">

</p>
<p>
Configure Teams from the Admin panel by selecting Agents > Teams > Add New Team. Rename your team and select "Create Team" to confirm. 
</p>
<br />

<p>
<img width="957" alt="Screenshot 2023-07-04 at 11 24 07 AM" src="https://github.com/technicallyjac/post-install-config/assets/138219436/b2d4dbd1-df1b-4681-bd7b-3da3579f9483">


</p>
<p>
Configure Agents from the Admin panel by selecting Agents > Add New Agents. Input the associate's information and assign appropriate permissions and team(s). Confirm changes.  
</p>
